About ngSpotify

Version: 2.4.1

ngSpotify Version 2.4.1 has been updated to Angular 14. With additional UI styling improvements.

The Spotify Web API went private in 2016. Since then user tokens expire every hour. User tokens will auto-renew.

Search for bands by name, or for musicians by name.
Tap/click the query list to display artist and albums, Level 2.
Tap/click "Album Details" for Level 3, track listings and audio previews.

Enjoy!
