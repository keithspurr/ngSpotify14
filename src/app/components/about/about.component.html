About ngSpotify

Version: 2.8.1

ngSpotify Version 2.8.1 has been updated to
Angular Version 18.

With additional UI styling improvements.

The Spotify Web API went private in 2016.
Since then API user tokens expire every hour.
ngSpotify's user tokens auto-renew.

Search for bands by name, or for musicians by name.
Tap/click the query list to display artist and albums, Level 2.
Tap/click "Album Details" for Level 3, track listings and audio previews.

Enjoy!
